How to install Free Web FLV Player by Web-Anatomy:

	1. Copy all files to your server
	2. Set proper CHMODS for both files (644) and folders (755)
	3. Run -> "http://your_domain/flv_player_free/"
	4. Run -> "http://your_domain/flv_player_free" and log in
	5. Select Installation link from top of Control Panel and follow remaining guide


That's all, If you have any problems visit http://web-anatomy.com/forum
